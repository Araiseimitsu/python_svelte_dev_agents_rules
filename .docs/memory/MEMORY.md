# MEMORY.md — 共有メモリ インデックス

全エージェント（Claude / Codex / OpenCode / Antigravity など）が共有する学びのインデックスである。
**タスク着手時（理解フェーズ）に必ず本ファイルを読み**、関連するカテゴリファイルを参照すること。
運用ルールは `AGENTS.md` §17「メモリ・自己成長」に従う。

## 書き方

- 1メモ = 1行。`- [カテゴリファイル名] 日付 — 要約（1行）` の形式で追記する。
- 詳細は各カテゴリファイルに書き、本ファイルには要約のみ置く。
- 古くなったメモは行ごと削除し、カテゴリファイル側も更新する。
- **インデックスは最大20行。** 超える場合は追記の前に統合・削除を行う（例外なし）。
